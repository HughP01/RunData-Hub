# Strava Data Loader And Visualiser
A simple Python project to fetch your running (and other activity) data from Strava, clean it, analyze performance trends, and create beautiful visualizations as well as AI powered coaching and sport analysis.
## Features
- Connect and authenticate with Strava
- Download all your activities (runs, rides, swims, etc.)
- Filter and focus on one specific sport 
- Calculate key metrics: distance, pace, moving time, elevation gain, heart rate trends
Generate insightful visualizations (weekly mileage, pace over time, heatmaps, etc.)
- Export cleaned data to CSV for further analysis
## Prerequisites
- Python 3.8+
- A Strava account
- A registered Strava API application (to get client_id and client_secret)
